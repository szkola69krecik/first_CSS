Pytania dotyczące stylu zewnętrznego

  1. Składnia teoretyczna

                    <style>
                .zewnętrzny-styl {
                    /* Definicje stylu zewnętrznego */
                }
                </style>

  2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik jest za to odpowiedzialny oraz ile razy może on występować w dokumencie, jako podłączenie stylu CSS?

    Styl zewnętrzny jest wstawiany w sekcji <head> dokumentu HTML za pomocą znacznika <link>. Może występować wiele razy w dokumencie, umożliwiając podłączenie wielu arkuszy styli CSS.

  3. Jak jest najprostsza forma wstawiania stylu zewnętrznego

                    <link rel="stylesheet" type="text/css" href="nazwa_pliku.css">

    Najprostszą formą wstawiania stylu zewnętrznego jest użycie znacznika <link> z atrybutami rel="stylesheet", type="text/css" i href="nazwa_pliku.css".

  4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasada musi być zachowana przy większej ilości dołączanych styli?

    Może być dołączonych wiele arkuszy styli do jednego dokumentu HTML. Aby zachować zasadę kaskadowości, należy pamiętać, że style z późniejszych arkuszy mają wyższy priorytet niż style z wcześniejszych arkuszy, chyba że są użyte selektory o większej specyficzności.

  5. Co się dzieje w przypadku konfliktów?

    W przypadku konfliktów, czyli gdy te same style są definiowane w różnych arkuszach styli podłączonych do tego samego dokumentu, zastosowane zostaną style z arkusza o wyższym priorytecie. Jeśli style mają taką samą specyficzność, zastosowane zostaną style z arkusza, który jest podłączony później w dokumencie.

  6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    Pliki arkuszy mogą być tworzone i edytowane za pomocą zwykłego edytora tekstowego, takiego jak Notepad, Notepad++, Sublime Text itp. Pliki te mają rozszerzenie ".css" i zawierają definicje stylów w postaci reguł CSS.

  7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    Specjalne edytory CSS, takie jak Adobe Dreamweaver, Visual Studio Code z rozszerzeniem CSS, czy Adobe Brackets, oferują bardziej zaawansowane funkcje edycji i podpowiedzi składni CSS, co ułatwia pracę z arkuszami styli. Można w nich tworzyć i edytować pliki arkuszy CSS w bardziej intuicyjny sposób.

  8. Jakie rozszerzenie posiadają pliki będące zewnętrznymi arkuszami styli?

    Pliki będące zewnętrznymi arkuszami styli mają rozszerzenie ".css".

  9. Jakie są zalety zewnętrznego arkusza styli?

    • Możliwość ponownego wykorzystania stylów w różnych dokumentach HTML.
    • Łatwość utrzymania i zarządzania stylami, ponieważ zmiany w arkuszu stylów mają wpływ na wszystkie strony, które go używają.

  10. Uwagi na temat pisania w CSS

    Przy pisaniu w CSS warto pamiętać o:

    • Używaniu selektorów o odpowiedniej specyficzności, aby precyzyjnie określić, do których elementów mają być zastosowane style.
    • Dodawaniu odpowiednich komentarzy, aby ułatwić zrozumienie kodu i jego późniejsze utrzymanie.
    • Dostosowywaniu kodu dla różnych przeglądarek, jeśli wymaga tego zgodność z różnymi silnikami renderującymi.
