Pytania dotyczące stylu lokalnego

  1. Gdzie jest wstawiany?

    Styl lokalny jest wstawiany w sekcji <style> wewnątrz znacznika <head> dokumentu HTML.

  2. Składnia teoretyczna (w ramce, konieczne wcięcia)

                    <style>
                .lokalny-styl {
                    /* Definicje stylu lokalnego */
                }
                </style>

    W ramce <style> umieszczamy definicje stylu lokalnego. Wcięcia są konieczne, aby oddzielić definicje od reszty kodu.

  3. Jakie mogą być selektory?

    Selektory w stylu lokalnym mogą być różnymi selektorami CSS, takimi jak nazwa elementu, klasa, identyfikator lub selektor pochodny. Można używać różnych kombinacji selektorów, aby precyzyjnie określić, do jakich elementów ma być zastosowany styl lokalny.

  4. Jakie mogą być cechy i wartości?

    "Cechami" w deklaracji stylu lokalnego mogą być różne właściwości CSS, takie jak kolor, tło, czcionka, marginesy itp. Każda cecha ma swoje właściwe wartości. Na przykład, dla właściwości color (kolor), wartościami mogą być nazwy kolorów (np. red, blue), wartości heksadecymalne (np. #FF0000) lub wartości RGB (np. rgb(255, 0, 0)).
