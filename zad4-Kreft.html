5 Sposobów podłączenia CSS do strony WWW

1. Dołączenie CSS do nagłówka HTML:


W tym przykładzie plik stylów o nazwie style.css jest dołączany do nagłówka HTML za pomocą elementu "link".


2. Dołączenie CSS bezpośrednio w kodzie HTML:


W tym przypadku kod CSS jest umieszczany bezpośrednio w sekcji "style" w nagłówku HTML.


3. Dołączenie CSS do tagu HTML:


W tym przypadku styl jest stosowany bezpośrednio do elementu HTML za pomocą atrybutu "style".


4. Dołączenie CSS w pliku HTML z wykorzystaniem tagu "style":


W tym przypadku kod CSS jest umieszczany w sekcji "style" w kodzie HTML.


5. Dołączenie CSS w pliku HTML z wykorzystaniem tagu "script":


W tym przypadku kod CSS jest umieszczany w sekcji "script" z atrybutem [type="text/css"]. Jednak ta metoda jest mniej popularna i zalecana tylko w wyjątkowych przypadkach.
